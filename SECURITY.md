# Security Policy

If you believe you've found a security vulnerability in PDFforge, please report it
privately rather than opening a public issue:

- Use [GitHub's private vulnerability reporting](https://github.com/louisschumacher-ops/PDFforge/security/advisories/new)
  for this repository, or
- Email the maintainer directly (see the GitHub profile at
  [louisschumacher-ops](https://github.com/louisschumacher-ops)).

Please include steps to reproduce, the affected version, and the potential impact.
We'll acknowledge reports as quickly as possible and coordinate a fix before any public
disclosure.
